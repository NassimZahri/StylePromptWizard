Image Prompt Generator API

Overview

This API allows you to generate descriptive prompts from images that can be used with image generation models like DALL-E, Midjourney, or Stable Diffusion.

Endpoints

Generate Prompt from Image

URL: /generate
Method: POST
Content-Type: multipart/form-data

Request Parameters

Parameter Type Required Description
image File Yes (if no image_url) The image file to analyze. Supported formats: PNG, JPG, JPEG, GIF, WEBP
image_url String Yes (if no image) A URL pointing to an image to analyze
style String No The style of prompt to generate:
  • detailed (default): Detailed description of the image
  • midjourney: Prompt formatted for Midjourney
  • dalle: Prompt optimized for DALL-E

Example Request

curl -X POST https://your-api-url/generate \
    -H "Accept: application/json" \
    -F "image=@/path/to/image.jpg" \
    -F "style=midjourney"

Response

JSON response with status and prompt: 

{
    "status": "success",
    "prompt": "Sleek silver Porsche 911 on coastal highway, dramatic sunset, aerial shot, reflective surface, motion blur, cinematic, professional car photography, ultra detailed, f/2.8 lens --ar 16:9 --v 6"
}

Error Handling

If an error occurs, the API will return a JSON response with a status code of 400 or 500: 

{
    "status": "error",
    "error": "Error message description"
}
